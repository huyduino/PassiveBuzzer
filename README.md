# Passive Buzzer

**Passive Buzzer** project shows how to use Passive Buzzer. By sending a pulse at different Hz, it will make different sound. In this experiment, it will play a short melody.

Author: **Huy Dang**

### Component Required

- (1) x Elegoo Uno R3 
- (1) x Passive buzzer
